Architecture

In the context of Backstage.io, which core component acts as the centralized system of record for all software assets (microservices, libraries, data pipelines, websites, ML models) within an organization?

close
A Software Templates
Your Answer
check
B Software Catalog
Correct Answer
C TechDocs
D Kubernetes Plugin

lightbulb Explanation

The correct answer is B) Software Catalog.

The Software Catalog is the core feature of Backstage that keeps track of ownership and metadata for all the software in your ecosystem (services, websites, libraries, data pipelines, etc.). It acts as a single system of record, enabling teams to manage and discover assets effectively.

  • A is incorrect because Software Templates are used for creating new projects rapidly using predefined boilerplates, not for acting as a system of record.
  • C is incorrect because TechDocs is the documentation-like-code feature within Backstage.
  • D is incorrect because the Kubernetes Plugin provides visibility into Kubernetes cluster health, not general cataloging.
