# Recommended CBA Stitch Prototype Export

This folder contains the clean recommended learner-facing screens downloaded from Stitch.

Source project: https://stitch.withgoogle.com/projects/15695885461367912017

Use the screens listed in `manifest.json`. Older screens in the parent Stitch export folder or in Stitch project history may contain stale prototype content and should be ignored.

Each screen folder contains:

- `code.html` — exported Stitch HTML
- `screen.png` — exported screenshot when available
- `metadata.json` — Stitch screen id and basic metadata
